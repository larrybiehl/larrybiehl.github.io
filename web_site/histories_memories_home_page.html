You can view the pictures in two different formatted screens controlled by a drop down list:

  • The "with list" mode includes a list of the pictures on the left side of the browser window. The scrollable list allows one to select a picture and display it directly.
  • The "no list" mode does not include the list of pictures. This mode was created for use in presentations.

The navigating buttons above a picture allow one to:

  • The ^ button allows one to return to this page.
  • Go to the first picture: <<; [or use the up arrow key]
  • Go to the previous picture in order: <; [or use the left arrow key]
  • Go to the next picture in order: >; [or use the right arrow key]
  • Go to the last picture >>; [or use the down arrow key]

The "Small, Medium, Large, Fit to Window, Fit to Window - no caption & Full" drop-down list allows one to control the size of the picture. "Fit to Window" is the default picture size.

  • "Small" causes a picture of around 300 pixels by 440 pixels to be displayed. "Small" is the best size to use on a 15 inch monitor.
  • "Medium" causes a picture of around 400 pixels by 600 pixels to be displayed. "Medium" is a good size to use on a 17 inch monitor.
  • "Large" causes a picture of around 600 pixels by 900 pixels to be displayed. "Large" is a good size to use on a 19 inch monitor.
  • "Fit to Window" causes a picture to be displayed that fully fills the web browser window. Internet Explorer expects the web browser window to take the full screen. The results of this vary with the web browser be used. Space is allowed below the picture for 3-4 lines of caption. This was created for presentions.
  • "Fit to Window-no caption" causes a picture to be displayed that fully fills the web browser window; no caption is displayed. Internet Explorer expects the web browser window to take the full screen. This was created for presentions.
  • "Full" causes the picture to be displayed at its full resolution. Most times this will be larger that the screen and one will have to scroll around the image.

The "Slide Show" drop-down list allows one to display the pictures in order by the selected time interval in seconds. The timer starts after the current picture has been fully displayed.

Information about the pictures:

Most pictures were scanned at 300 pixels per inch. Small pictures were scanned at 600 pixels per inch. Slides were scanned at 1200 pixels per inch. Each original picture contains around 1700 x 1200 pixels and takes up 6 megabytes of disk space. The pictures in this presentation represent a jpeg compressed version of the original with a compression setting of "75-90".
